FaQ

Perguntas Frequentes

Como posso adquirir bilhetes para os eventos presentes no website?

Para adquirir os bilhetes basta nos contactar pelo nosso formulário que nós iremos proceder à venda.

O site é para ficar online por muito tempo ou é um projeto apenas a curto prazo?

Por enquanto este site é um projeto escolar, ou seja, quando for apresentado vai deixar de estar online, mas quem sabe futuramente ele poderá ficar online denovo.

O site foi construído por uma equipa de desenvolvedores ou apenas por um desenvolvedor?

Como referi anteriormente este projeto foi feito no âmbito de um projeto final de curso, por isso, foi desenvolvido apenas por uma pessoa só.
